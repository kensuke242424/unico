# OriginalApp
オリジナルアプリケーション「名前未定」
個人で音楽、絵、グッズ、雑貨など製作をしている販売者が使えるアプリを開発中です。
以下の機能を実装予定です。

・アイテムのコレクションライブラリ機能
・アイテム在庫の増減管理機能
・アイテムの売上計上及び閲覧機能
